LCIDPlay
========
A CSharp program that converts a screen capture of the MSDN web site on MSLCID into a CSV file.
See http://msdn.microsoft.com/en-us/library/cc233965.aspx for more details.
Also see the Bing Translation REST service.

NOTE: clone creates a new directory in the current directory.
CD to the GitHub directory for creating new projects

git clone https://github.com/BradThompson/LCIDPlay.git
